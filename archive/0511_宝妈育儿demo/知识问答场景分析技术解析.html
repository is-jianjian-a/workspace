3️⃣Step 3: 内容质量评估

UGC vs 营销内容识别
基于营销信号和UGC信号，将内容分为4类，筛选高可信度内容
# 内容类型判断规则 marketing_signals = ['好物', '推荐', '测评', '必买', '清单', '品牌', '链接', '优惠', '团购', '种草', '带货', '广告', '赞助'] ugc_signals = ['求助', '怎么办', '正常吗', '崩溃', '焦虑', '后悔', '吐槽', '日记', '记录', '日常', 'vlog', '我的', '我家', '我娃'] def classify_content(text): has_marketing = any(s in text for s in marketing_signals) has_ugc = any(s in text for s in ugc_signals) if has_marketing and not has_ugc: return '纯营销' elif has_ugc and not has_marketing: return '纯UGC' elif has_marketing and has_ugc: return '混合' else: return '中性'
