|-----|---------|

## 快速判断流程

```
1. 读取UGC标题+内容
2. 检查是否含创造者关键词
   ├─ 是 → 检查是否含消费者关键词
   │       ├─ 否 → 排除（纯创造者）
   │       └─ 是 → 排除（混合视角，保守策略）
   └─ 否 → 检查是否含消费者关键词
           ├─ 是 → 保留（纯消费者）
           └─ 否 → 人工判断或根据二层主题决定
```
