# 报告规范 v3.1+

> 适用：ai-demand-research-v3 及后续 v3.x 版本
> 核心改进：双层主题标签 + UGC级多维标签 + 可展开折叠交互

---

## 1. 双层主题标签体系

### 1.1 设计目的

解决单层主题粒度太粗、无法指导具体产品方向的问题。通过两层标签实现：
- **一层主题**：宏观需求分类，用于报告导航和概览
- **二层主题**：细粒度功能点，用于指导具体产品特性设计

### 1.2 层级定义

| 层级 | 粒度 | 数量 | 作用 | 示例 |
|------|------|------|------|------|
| **一层主题** | 粗粒度功能领域 | 8-15个 | 宏观需求分类、报告导航 | 视频生成/编辑、学习/笔记 |
| **二层主题** | 细粒度功能点 | 每个一层下2-5个 | 具体产品方向、需求细化 | AI生成视频、自动剪辑、笔记工具 |

### 1.4 规则

- 每条UGC可同时拥有**多个**一层和二层标签（多标签分类）
- 二层主题不必严格从属于某一层主题，允许跨主题关联
- 标签提取采用"首次匹配即命中"策略，避免过度细分

## 2. UGC级多维标签

### 2.1 四个维度

每条UGC在报告中必须展示以下四个维度的标签：

| 维度 | 标签颜色 | 配色变量 | 说明 |
|------|---------|---------|------|
| **平台** | 紫色系 | `#a5b4fc` / `rgba(99,102,241,0.15)` | xhs / bilibili / douyin / kuaishou / zhihu |
| **情绪** | 黄色系 | `#fcd34d` / `rgba(245,158,11,0.15)` | frustration / 焦虑/急迫 / 困惑/迷茫 / 疲惫/厌倦 / 期待/渴望 / 惊喜/兴奋 |
| **场景** | 绿色系 | `#6ee7b7` / `rgba(16,185,129,0.15)` | 通勤/路上 / 工作/学习 / 睡前/休息 / 吃饭/碎片 / 运动/健身 / 家庭/陪伴 |
| **二层主题** | 红色系 | `#fca5a5` / `rgba(239,68,68,0.15)` | 该UGC归属的细粒度功能点 |

### 2.3 展示规则

- 标签显示在UGC文本**下方**，用不同颜色区分
- 标签旁显示该标签在当前主题下的出现次数（如 `期待/渴望(29)`）
- **无标签的维度可隐藏**，不显示空标签区域
- 标签过多时（>5个），优先展示高频标签

## 3. 可展开折叠的UGC列表

### 3.1 交互设计

- 每个一层主题为一个**可折叠卡片**
- 卡片头部显示：
  - 主题名（大号字体）
  - UGC数量
  - 平均互动量
  - 平台/情绪/场景/二层标签统计（标签云形式）
- 点击卡片头部**展开/收起**该主题下的所有UGC
- 展开时显示平滑动画（`max-height transition`）
- 收起时旋转箭头回正
- UGC按**互动量+质量分**排序，高价值内容优先展示

### 3.2 视觉规范

```css
/* 色彩系统 */

  --bg-secondary: #12121a;

```

- **深色主题**（`#0a0a0f` 背景）
- 卡片hover时有**边框高亮**效果（`border-color: var(--accent)` + `box-shadow`）
- 展开状态有**旋转箭头**指示（`transform: rotate(90deg)`）
- **响应式布局**（移动端适配，flex-wrap）

### 3.3 动画实现

```javascript
function toggleTheme(id) {
    const list = document.getElementById(id);
    const header = document.getElementById('header-' + id);
    if (list.classList.contains('open')) {
        list.classList.remove('open');
        header.classList.remove('expanded');
    } else {
        list.classList.add('open');
        header.classList.add('expanded');
    }
}
```

## 5. 与 v3.0 的差异

| 维度 | v3.0 | v3.1+ |
|------|------|-------|
| 主题层级 | 单层（信号） | 双层（一层+二层） |
| UGC展示 | 仅代表性样本 | 全量可展开折叠 |
| UGC标签 | 无 | 平台+情绪+场景+二层主题 |
| 主题统计 | 仅提及次数 | 平台/情绪/场景/二层分布 |
| 交互 | 静态卡片 | 可展开折叠 + 动画 |
