## 三、核心分歧分析

### 3.1 LLM的判定逻辑

LLM将这些内容归为D(观看视频)的依据：
- 搜索词包含"看视频"、"刷视频"
- 内容提到了"手机"、"玩手机"、"刷手机"
- 讨论了使用手机带来的负面体验
- 涉及"沉迷"、"焦虑"、"眼睛不舒服"等与观看行为相关的症状

### 3.2 用户的判定逻辑

用户将这些内容改为A(视频无关)的依据：
- 内容核心是**"手机使用问题"**而非**"视频观看问题"**
- 讨论的是**泛化的手机沉迷/焦虑**，不特指视频
- 涉及的是**心理健康、身体健康、时间管理**等更广泛主题
- **"玩手机" ≠ "看视频"**，口语中"玩手机"泛指使用手机的各种行为（社交、游戏、浏览等）
- 即使提到"刷手机"，内容焦点在**成瘾机制/心理分析**，而非**观看体验/功能需求**

### 3.3 关键分歧点


## 四、经验总结

### 4.1 LLM分类的系统性偏差

**偏差类型**: "搜索词锚定效应"

LLM过度依赖搜索词来判断内容类别，当搜索词包含"看视频"、"刷视频"时，LLM倾向于将所有提到"手机"+"负面体验"的内容归为D类，而忽略了内容本身的主题焦点。

**具体表现**:
- D类144条中，有**24条**包含"手机"但不包含任何明确视频观看词（如"视频"、"看剧"、"追剧"、"电影"、"B站"、"抖音"等）
- 这24条中，用户实际修改了8条（33.3%），说明还有16条可能存在类似误判

### 4.2 分类规则改进建议

#### 规则1："玩手机"≠"看视频"

```
当内容提到"玩手机"但满足以下任一条件时，应归为A(视频无关)：
✓ 内容焦点是手机成瘾、时间管理、心理健康
✓ 不特指视频，而是泛指手机使用
✓ 讨论的是"为什么停不下来玩手机"而非"看视频的体验如何"
```

#### 规则2：明确视频相关词的判定标准

```
内容必须包含至少一个明确的视频观看词，才能归为D：
- 平台词：B站、抖音、快手、小红书、视频号、YouTube
- 内容词：视频、看剧、追剧、刷剧、电影、长视频、短剧、综艺
- 行为词：看视频、刷视频、追剧、观影

如果内容只提到"手机"、"玩手机"、"刷手机"而无上述词，
且讨论的是成瘾/健康/心理主题 → 归为A
```

#### 规则3：主题焦点优先于搜索词

```
分类时应优先判断内容的主题焦点，而非搜索词：
- 焦点=视频观看体验/需求/功能 → D
- 焦点=手机成瘾机制/心理健康/身体健康 → A
- 焦点=视频创作技巧/工具 → B
- 焦点=推荐某个视频 → C
```

### 4.3 搜索词设计反思

**问题搜索词**: "手机看视频 缺点"

这个搜索词的问题：
1. "手机看视频"在口语中常被简化为"玩手机"
2. "缺点"一词过于宽泛，容易引出泛化的手机使用问题
3. 用户讨论"手机缺点"时，往往不区分是看视频还是其他行为

**改进建议**:
- 避免使用"手机看视频"这种口语化表达
- 使用更精确的搜索词，如"看视频 眼睛累"、"刷视频 上瘾"
- "缺点"可替换为更具体的词，如"痛点"、"问题"、"体验"
