# 搜索词设计：宝爸宝妈育儿收集需求

## 设计原则
- **正交搭配**：场景词 × 功能/痛点/体验词，不预设具体功能方向
- **系统层面优先**：从手机系统层面打捞诉求，非APP功能层面
- **全周期覆盖**：备孕 → 孕期 → 产后 → 育儿 → 早教

## 第一阶段：泛化搜索（场景发现）

| 搜索词 | 类型 | 覆盖笔记 | 关注焦点 |
|--------|------|---------|---------|
| 孕期 | 场景+状态 | 105 | 孕期知识、注意事项 |
| 产后 | 场景+状态 | 104 | 产后恢复、情绪、育儿适应 |
| 早教 | 场景+行为 | 101 | 早教知识、方法、资源整理 |
| 备孕 | 场景+行为 | 100 | 备孕知识、准备、记录 |
| 育儿 | 场景词 | 98 | 育儿知识、经验泛化表达 |

## 搜索词评审检查点
- [x] 覆盖"做什么→遇到什么→情绪如何→怎么办"完整链路
- [x] 无同质化词（各词语义区分明显）
- [x] 采用正交搭配，避免预设功能方向
- [x] 从手机系统层面出发，非APP功能层面
- [x] 覆盖备孕到早教全周期
